H1 até o H6

Essas são as tags usadas para definir os títulos e subtítulos em HTML. h1 é o maior e h6 é o menor.

Voltar

P

Essa tag é usada para definir parágrafos de texto.

Voltar

Mark

Essa tag é usada para destacar partes de texto, geralmente aplicando um fundo colorido.

Voltar

Small

Essa tag é usada para definir texto com um tamanho menor, indicando que é um texto de menor importância.

Voltar

i

Essa tag é usada para aplicar estilo de itálico ao texto.

Voltar

Strong

Essa tag é usada para definir texto fortemente enfatizado, exibido em negrito.

Voltar

ol

Essa tag é usada para criar listas ordenadas, onde cada item é numerado.

Exemplo:

  1. Item 1
  2. Item 2
  3. Item 3
Voltar

ul

Essa tag é usada para criar listas não ordenadas, onde cada item é marcado com um ponto.

Exemplo:

  • Item 1
  • Item 2
  • Item 3
Voltar

li

Essa tag é usada para definir um item de lista, seja em uma lista ordenada "ol" ou não ordenada "ul".

Voltar

a

Essa tag é usada para criar links. O atributo href especifica o destino do link.

Voltar

hr

Essa tag é usada para criar uma linha horizontal, muitas vezes usada para separar seções em uma página.

Voltar

sub

Essa tag é usada para representar texto que deve ser exibido como subscrito (abaixo da linha de base).

Voltar

sup

Essa tag é usada para representar texto que deve ser exibido como sobrescrito (acima da linha de base).

Voltar

blockquote

Essa tag é usada para definir um

bloco de citação.
